COMPUTER

A computer is an electronic device,operating under the controlled instruction stored in its own memory that can accept data input,process the data according to the specified rule,produce the information output,and save information for future use.



Functionalities Of Computer 2

Any digital computer carries out five function in a gross term:

  • take data as input.
  • process the data.
  • generates the output.


Computer Components

Any type of computer contains HARDWARE and SOFTWARE.
